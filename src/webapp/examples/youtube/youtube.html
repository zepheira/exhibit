Youtube: Rendered from XML Data

This page demonstrates the XML importer. The link in the head applies three tags.
  • ex:itemTags specifies which tags in the XML document should be taken as the (roots of) items. The tags are assumed to name the types of the items.
  • ex:labelProperties specifies, for each item type, the property of that item that should be used as the label for the item (its value is copied to the label property
  • ex:parentRelations is used if one item Y may be a descendant of another item X in the XML; it specifies, for each type, the name a the relation that should be used to connect the parent to its descendant in the generated data.
